# 🎓 School Management System

A modern **School Management System** built using Vue 3, PrimeVue, Tailwind CSS and Node.js.  
This system helps manage teachers, students, and administrative operations efficiently.

---

## ✨ Features

- 📚 Teacher Management (Add, Edit, Delete)
- 👨‍🎓 Student Management
- 🔍 Search & Filter functionality
- 📊 Dynamic Data Tables
- 🧩 Modular Component Structure
- 🎨 Modern UI with Tailwind CSS
- ⚡ Fast & Responsive Design

---

## 🛠 Tech Stack

### Frontend
- Vue 3 (Composition API)
- PrimeVue
- Tailwind CSS
- Vue Router
- Pinia (State Management)

### Backend
- Node.js
- Express.js
- MongoDB
